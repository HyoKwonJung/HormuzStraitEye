# Dashboard Prototype Quick Start

Run the commands below to preview the dashboard immediately.

```bash
cd app
python3 -m http.server 4173
```

Then open `http://localhost:4173` in your browser.

## What's included
- Hero status cards (Computed Risk, Official Threat, Incidents, Warnings)
- Live operational map (Leaflet + OpenStreetMap tile layer)
- Last 24h timeline
- Threat breakdown bars
- "Why Risk Is Up" explanation panel

> This is a visual prototype before backend/API integration. Values shown are sample data.
